# Data Ownership and Portability

## Non-Negotiable Platform Requirement
Users own their data.

The platform must always support:
- Easy Export
- Full Backup
- Migration
- Import
- No Vendor Lock-in

## Architecture Guardrails
- Standalone workflows cannot depend on proprietary integration availability.
- Export/backup formats should remain documented and practical for re-use.
- Migration/import pathways should preserve core business context (documents, profiles, payments, timelines, and preferences).
