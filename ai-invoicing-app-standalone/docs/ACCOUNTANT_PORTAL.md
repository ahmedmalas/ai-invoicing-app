# Accountant / Bookkeeper / BAS Agent Portal

## Purpose
Provide secure advisor collaboration without shared passwords.

## Access Model
- User invites advisor with permission-based access.
- Access scopes can limit data visibility and actions.
- All advisor actions are recorded in audit logs.

## Advisor Capabilities
- View invoices, expenses, receipts, bills, GST/BAS summaries, spending analytics, supplier documents, reports, audit logs, and attachments.
- Request missing documents and leave notes.
- Flag issues and review reconciliation outcomes.
- Export reports as permitted.

## Security Principles
- No shared credentials.
- Least-privilege permissions.
- Timeline and audit traceability for all advisor interactions.
