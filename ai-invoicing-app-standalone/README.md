# Ai Invoicing App (Standalone)

This repository contains the standalone foundation for the **Ai Invoicing App**.

## Core Product Goal

Build a standalone AI-assisted invoicing app with unlimited options and customization for:

- Invoice layouts
- Customer profiles
- Line items
- GST handling
- Payment details
- Invoice numbering
- PDF export
- Reusable templates
